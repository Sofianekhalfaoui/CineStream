# 🎬 Cinematic

تطبيق شامل لمشاهدة الأفلام والمسلسلات

## 🚀 الميزات

- 🎥 مشاهدة الأفلام والمسلسلات
- 📱 تطبيق جوال (Android و iOS)
- 🌙 واجهة أنيقة وسهلة الاستخدام
- 🔐 تسجيل آمن
- 💾 حفظ المفضلة

## 🛠️ التقنيات المستخدمة

- React 19 + TypeScript
- Vite (Build Tool)
- Tailwind CSS
- Firebase
- Capacitor (لتحويل الويب إلى تطبيق موبايل)
- Plyr (مشغل الفيديو)

## 📦 التثبيت والتشغيل

### متطلبات المشروع
- Node.js 18+
- npm أو yarn

### التشغيل المحلي
```bash
npm install
npm run dev
```

### بناء التطبيق
```bash
npm run build
```

### تحويل إلى APK
```bash
npm run cap:sync
npm run cap:open
```

ثم افتح المشروع في Android Studio وقم ببناء APK.

## 📱 بناء APK تلقائياً

يتم بناء APK تلقائياً عند الدفع إلى فرع `main` من خلال GitHub Actions.
